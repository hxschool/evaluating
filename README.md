# EvaluatingSystem
综合素质测评管理系统

<<<<<<<<<<<<<<<<综合素质测评系统>>>>>>>>>>>>>>>>

系统安装目前需要以下步骤：（手动安装）

1.导入sql文件到服务器的数据库（mysql)

2.将项目文件使用Eclipse等IDE打开

3.配置WEB-INF目录中的dbconfig.properties文件（需要配置服务器上的数据库信息）

4.导出项目war包，发布到Tomcat中。

sql文件中已经有一些测试数据，可以直接测试。
